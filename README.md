# Telegram 3D Girl App

Простое Telegram Mini App с 3D моделью девушки.

## Описание

Это простое веб-приложение, которое отображает 3D модель девушки, созданную из примитивов Three.js. Модель можно вращать, перетаскивая мышью или пальцем на сенсорных устройствах.

## Особенности

- Статическая 3D модель девушки (без анимаций)
- Интерактивное вращение модели
- Поддержка сенсорных устройств
- Адаптивный дизайн
- Готово для использования в Telegram Mini App

## Установка и запуск

1. Откройте `index.html` в браузере
2. Или используйте локальный сервер:
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```

## Размещение на веб-сервере

Telegram Mini App требует HTTPS. Вот несколько бесплатных вариантов:

### Вариант 1: GitHub Pages (рекомендуется)

1. Создайте репозиторий на [GitHub](https://github.com)
2. Загрузите все файлы в репозиторий
3. Перейдите в Settings → Pages
4. Выберите ветку `main` и папку `/ (root)`
5. Сохраните - через минуту ваше приложение будет доступно по адресу:
   `https://ваш-username.github.io/название-репозитория`

### Вариант 2: Netlify (самый быстрый)

1. Зарегистрируйтесь на [netlify.com](https://www.netlify.com)
2. Перетащите папку с файлами в область "Drag & drop"
3. Получите автоматический HTTPS URL (например: `https://random-name-123.netlify.app`)

### Вариант 3: Vercel

1. Зарегистрируйтесь на [vercel.com](https://vercel.com)
2. Нажмите "Add New Project"
3. Загрузите файлы или подключите GitHub репозиторий
4. Получите HTTPS URL

### Вариант 4: Для локальной разработки (ngrok)

1. Установите [ngrok](https://ngrok.com)
2. Запустите локальный сервер: `python -m http.server 8000`
3. В другом терминале: `ngrok http 8000`
4. Используйте полученный HTTPS URL

## Использование в Telegram

1. Загрузите файлы на один из веб-серверов выше (получите HTTPS URL)
2. Создайте бота через [@BotFather](https://t.me/BotFather) в Telegram
3. Используйте команду `/newapp` для создания Mini App
4. Укажите HTTPS URL вашего приложения
5. Готово! Ваше приложение будет доступно через бота
